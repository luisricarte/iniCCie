IniCCie

    A objetivo do iniCCie é que seja possível professores deixarem de usar o email para divulgar vagas de P&D, PIBIC, PET, MONITORIA, entre outras e passem a utilizar o iniCCie. Então a aplicação contará com:
  • Sistema de CRUD de Vagas
  • Sistema de CRUD de Aluno
  • Sistema de CRUD de Professor

    • Além disso, teremos as interações:
    • Uma vaga pode ter muitos alunos cadastrados
    • Um professor poderá cadastrar muitas vagas
    • Um aluno poderá se candidatar para uma ou muitas vagas
    • Um aluno poderá cancelar a sua inscrição na vaga
    • Um professor poderá encerrar, editar ou criar uma vaga
